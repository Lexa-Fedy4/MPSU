## Практика

Рекомендуемая среда из методички: **microC PRO for PIC/AVR** и **Proteus 8 Professional**.

### Задание 1 — PIC16F84A

- Папка: `praktika/zadanie_01/` — код + `proteus_shema.md`
- Компилятор: **microC PRO for PIC**

### Задания 2–30 — ATmega328P

- Компилятор: **Atmel Studio** (avr-gcc)
- В проекте: `F_CPU` и частота кварца из `proteus_shema.md`

#### Как открыть в Atmel Studio + Proteus

1. Создайте проект GCC C → ATmega328P
2. Скопируйте `praktika/zadanie_XX/zadanie_XX.c` в проект
3. Укажите `F_CPU` (см. `proteus_shema.md`)
4. Соберите проект → загрузите `.hex` в Proteus

#### Семисегментные индикаторы (задания 2, 5, 8, 10, 12, 14, 18, 21, 23, 26, 29)

В mikroC **нет отдельного файла библиотеки** для 7-seg — используется стандартный подход:

1. **Таблица кодов** `seg_table[10]` — из **Tools → Seven Segment Editor**
2. **Delay_ms()** — встроенная функция mikroC
3. Разряды индикатора на **PC0/PC1** (PORTD), **PC0/PC1** (PORTB) или **PB0/PB1** (PORTC) — см. схему задания

#### Типы заданий

| Тип | Номера |
|-----|--------|
| Семисегментный индикатор | 2, 5, 8, 10, 12, 14, 18, 21, 23, 26, 29 |
| Кнопка + светодиод | 4, 6, 9, 11, 15, 19, 22, 25, 27, 28 |
| Два светодиода (таймеры) | 1, 3, 7, 13, 16, 17, 20, 24, 30 |
