# Общая инструкция Proteus 8

## Базовая схема ATmega328P (для заданий 2–30)

```
                    +5V
                     |
                    [10k]
                     |
    RESET(1) --------+-------- ATMEGA328P
    XTAL1(9) ----+                    |
    XTAL2(10) ---+--[CRYSTAL]--+      |
                 |              |      |
                [22pF]        [22pF]   |
                 |              |      |
                GND            GND    GND
    VCC(7), AVCC(20) -----------------+5V
```

## Поиск компонентов в Proteus

| Что нужно | Имя в библиотеке |
|-----------|------------------|
| МК AVR | ATMEGA328P |
| МК PIC | PIC16F84A |
| Индикатор | 7SEG-COM-CATHODE |
| Кнопка | BUTTON |
| Светодиод | LED |
| Резистор | RES |
| Конденсатор | CAP |
| Кварц | CRYSTAL |
| Питание | POWER, GROUND |

## Подключение LED (активный HIGH)

```
PORTx.y ----[220 Ом]----|>|---- GND
                         LED
```

## Подключение кнопки

```
PORTx.y ----[BUTTON]---- GND
```
В коде включена внутренняя подтяжка (PORTx.y = 1). Нажатие = лог. 0.

## Подключение 2×7SEG (мультиплексирование)

1. Сегменты A..G обоих индикаторов — **параллельно** на один порт (через 220 Ом).
2. COM каждого разряда — на **отдельный** вывод МК (см. `proteus_shema.md` в папке задания).
3. Тип индикатора: **общий катод** (7SEG-COM-CATHODE).

## Загрузка программы

1. Скомпилируйте `.c` в microC → Build → получите `.hex`.
2. Двойной клик на МК → **Program File** → выберите `.hex`.
3. **Clock Frequency** = частота из задания (8 / 10 / 16 МГц).
4. Нажмите Play (симуляция).
