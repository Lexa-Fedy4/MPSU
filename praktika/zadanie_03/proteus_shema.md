# Схема Proteus — задание 3

**МК:** ATmega328P | **Частота:** 10 МГц

## Компоненты

U1 ATMEGA328P, X1 CRYSTAL 10MHz, C1/C2 22pF, R1 10k, R2/R3 220, D1/D2 LED

## Питание и кварц

VCC(7), AVCC(20)→+5V | GND(8), AGND(22)→GND | RESET(1)→+5V через 10k | XTAL1/2 + кварц 10MHz

## Подключение

| Устройство | Цепь |
|------------|------|
| LED1 | **PC6** → 220Ω → LED → GND |
| LED2 | **PB5** → 220Ω → LED → GND |

## Запуск

Program File = hex из `zadanie_03.c`, Clock = 10 MHz.

## Поведение

LED1: 10с горит → 2с пауза → 5с горит → пауза → цикл.  
LED2: мигание 3с вкл / 3с выкл.
