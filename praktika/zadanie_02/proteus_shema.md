# Схема Proteus — задание 2

**МК:** ATmega328P | **Частота:** 10 МГц | **Компилятор:** microC PRO for AVR

## Компоненты

| Обозначение | Компонент Proteus |
|-------------|-------------------|
| U1 | ATMEGA328P |
| X1 | CRYSTAL 10MHz |
| C1, C2 | CAP 22pF |
| R1 | RES 10k |
| R2–R8 | RES 220 (сегменты) |
| DIS1, DIS2 | 7SEG-COM-CATHODE |

## Питание и кварц (ATmega328P)

| Вывод | Подключение |
|-------|-------------|
| 7 VCC, 20 AVCC | +5V |
| 8 GND, 22 AGND | GND |
| 1 RESET | +5V через R1 10k |
| 9 XTAL1, 10 XTAL2 | Кварц X1 + C1/C2 на GND |

## Семисегментники (2 шт., общий катод)

Сегменты **обоих** индикаторов соединены параллельно:

| Сегмент | → МК |
|---------|------|
| A | PD0 через 220Ω |
| B | PD1 через 220Ω |
| C | PD2 через 220Ω |
| D | PD3 через 220Ω |
| E | PD4 через 220Ω |
| F | PD5 через 220Ω |
| G | PD6 через 220Ω |

| Разряд | COM индикатора → МК |
|--------|---------------------|
| Десятки (левый) | **PC0** |
| Единицы (правый) | **PC1** |

> В Proteus выберите **7SEG-COM-CATHODE**. COM подключается напрямую к PC0/PC1.

## Запуск

1. Соберите схему, загрузите `.hex` из `zadanie_02.c`.
2. Clock Frequency = **10 MHz**.
3. На индикаторе: счёт **0 → 10 → 0**.

```
        PD0..PD6 ----[220R]---- A..G (оба 7SEG)
        PC0 ------------------- COM (десятки)
        PC1 ------------------- COM (единицы)
```
