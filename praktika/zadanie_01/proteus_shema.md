# Схема Proteus — задание 1

**МК:** PIC16F84A | **Частота:** 8 МГц | **Компилятор:** microC PRO for PIC

## Компоненты

| Обозначение | Компонент Proteus |
|-------------|-------------------|
| U1 | PIC16F84A |
| X1 | CRYSTAL 8MHz |
| C1, C2 | CAP 22pF |
| R1 | RES 10k (MCLR) |
| R2, R3 | RES 220 |
| D1, D2 | LED |
| SW1 | BUTTON |

## Подключение

### Питание и кварц
| Вывод МК | Куда |
|----------|------|
| VDD (14) | +5V |
| VSS (5) | GND |
| MCLR (4) | +5V через R1 10k |
| OSC1 (16), OSC2 (15) | Кварц X1, конденсаторы C1/C2 на GND |

### Светодиоды
| Цепь | Подключение |
|------|-------------|
| LED1 (RB6) | RB6 → R2 220Ω → анод D1 → катод D1 → GND |
| LED2 (RB3) | RB3 → R3 220Ω → анод D2 → катод D2 → GND |

### Кнопка
| Цепь | Подключение |
|------|-------------|
| SW1 | RA2 → кнопка → GND (подтяжка RA2 в коде) |

## Запуск в Proteus

1. Соберите схему по таблицам.
2. Скомпилируйте `zadanie_01.c` → получите `.hex`.
3. U1 → Edit Properties → **Program File** = ваш `.hex`.
4. **Clock Frequency** = 8 MHz.
5. Run simulation.

## Ожидаемое поведение

- LED1 горит, когда кнопка **не нажата**.
- LED2 мигает с периодом 1 с (0,5 с вкл / 0,5 с выкл).
